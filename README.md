An advanced, light-weight **26-in-1 Information Gathering and Vulnerability Scanning Framework** developed in Python for cybersecurity research, penetration testing simulation, and OSINT exploration.

## 🚀 Key Features

### 🌐 Network & Website Scanner
* **Website Vuln Scanner:** Audits missing critical security headers (CSP, X-Frame-Options).
* **Advanced Port Scanner:** Multithreaded engine to map open connection routes.
* **DNS & Subdomain Finder:** Maps hidden infrastructure assets quickly.

### 🕵️‍♂️ OSINT Tools
* **Username Tracker:** Cross-references handles across multiple mainstream platforms.
* **D0x Dossier Engine:** Structures and exports local analytical investigation identity records.
* **Email & Phone Intelligence:** Resolves regional metadata parameters for analysis.

### 🛠️ Utility Modules
* **Password Cracker Simulation:** Tests multi-threaded ZIP extraction dictionaries.
* **Crypto Handler:** Instantly generates cryptographic primitives (MD5, SHA-256).
* **Target IP Generator:** Allocates localized random network address simulation matrix layouts.
